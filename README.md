## Project 1. Analysis of characteristics of top performers within the company

- 문제 정의
   - 회사 내에서 업무 성과를 올리고 있는 직원들이 증가하는 추세를 보입니다. 그래서 회사 내에서 우수한 성과를 올리는 직원들이 나오는 요인들을 확인

- 수행역할
   - 데이터 전처리, EDA(탐색적 데이터 분석), 데이터 시각화, 데이터 타입 및 결측치 확인
   - 회사 내 데이터들 간의 상관관계 분석과 회귀분석을 통해 영향을 주는 요인을 파악
   - 분석 결과를 바탕으로 회사 인사팀 내부 회의를 열었으며 성과를 위한 지표, 평가 항목 점검 등을 제안
 
- 기대효과 및 Lesson and Learned
  + 기대효과
    - 프로젝트를 통해 성과 우수자가 될 수 있는 요인들을 파악하고 효율적인 인력활용 방안을 도출할 수 있음
    - 데이터 기반 방안 도출을 통해 비용 절감이 가능함
  + Lesson and Learned
    - 데이터 전처리 기법 습득
    - 상관관계 분석으로 인한 관계파악 의존성 줄이기 위한 회귀분석 추가 시행 
   
* 프로젝트 상세 및 코드 -> [바로가기](https://github.com/Y-Junghye/Portfolio/blob/8aa77da0a49cbf30cd009c7a87e6e350fc06a48e/Project%201.%20Top%20Performer%20Characteristics%20Analysis%20/2.%20Code/Project%20Code.ipynb)

## Project 4. Identifying Pass:Fail in the semiconductor process

- 문제 정의
   - 센서 데이터를 통해 반도체 공정에서 발생할 수 있는 수율 이탈의 주요 원인 파악

- 수행역할
   - 데이터 전처리, EDA(탐색적 데이터 분석), 데이터 시각화, 데이터 타입 및 결측치 확인
   - 데이터 전체 및 특정 변수에 대한 로지스틱 모델과 랜덤포레스트 모델 분석을 통해 중요 변수 파악
 
- 기대효과 및 Lesson and Learned
  + 기대효과
    -  반도체 제조과정의 품질을 기존보다 20% 이상 향상
    -  반도체 제조 공정 효율성이 높아짐과 동시에 품질도 향상되면서 소문을 들은 신규고객이
       이전보다 30% 증가함.
    -  신규 고객 증가로 월 수익이 5억 증가함.
      
  + Lesson and Learned
    - 반도체 제조 공정과정에서 효율성을 높이기 위해 불량여부(합격/불합격) 판단이 핵심사항임을 습득함
    - 올바른 데이터 전처리 과정이 불량여부와 이상데이터 판단으로 이어지며 매출에도 큰 영향을 미치기 때문에
     데이터 전처리 과정을 좀더 깊이있게 학습하고 실무에 적용시킬 필요 있음.

## Project 5. Sales Analysis by Category

- 문제 정의
   - 회사의 매출 관점에서 카테고리별(대카테고리, 서브 카테고리) 매출을 분석하고 향후 성장 방안 탐색필요

- 수행역할
   - 2014년 3월 기준 온라인 커머스 기업의 데이터 수집
   - 데이터 전처리, EDA(탐색적 데이터 분석), 데이터 시각화, 데이터 타입 및 결측치 확인
     (연도별, 월별, 요일별, 고객구매처별 구매 경향 분석)
   - 각 카테고리별 회귀분석 진행 및 상위 3개 카테고리 분석
 
- 기대효과 및 Lesson and Learned
  + 기대효과
    -  카테고리별 매출액 구분 및 매출성장방안 파악
    -  도시별, 연도별, 월별, 요일별, 고객구매처별 등 다양한 구분으로 고객상품소비 특성분석 가능
      
  + Lesson and Learned
    - 하나의 고객 데이터를 다양한 집단(도시, 연도, 월, 요일 등)으로 나누어 분석을 하면, 일반적인 데이터
     분석에서는 파악할 수 없는 인사이트 도출 가능함을 습득

## Project 6. Establishment of additional customer purchases

- 문제 정의
   - 2021년의 '다잇다' 고객의 구매 이력 데이터을 이용해 고객의 추가구매 유도방안 수립 필요

- 수행역할
   - 2021년의 '다잇다' 고객의 데이터 수집(2015~2017년 데이터)
   - EDA(탐색적 데이터 분석) 및 데이터 정제, 데이터 시각화, 데이터 결측치 등 확인과 가공
   - 일반 데이터 분석 및 클러스터링 분석 진행
 
- 기대효과 및 Lesson and Learned
  + 기대효과
    -  월별, 요일별(주중과 주말), 연령대별(10~60대 이상), 연도별 분석을 통한 고객구매경향 분석 및
       추가 구매유도 위한 방안 마련 가능
    - 클러스터링 분석으로 유사 군집별 소비성향 파악 가능
      
  + Lesson and Learned
    - 이커머스 데이터 분석에서 월별, 고객별 등 분석 이외에도 유사 그룹별로 묶어 분석을 진행할 경우 유사
      그룹 내에서도 고객들의 선호도가 달라짐을 인지하였음.

## Project 7. Analyze customer course-taking patterns across different types of courses

- 문제 정의
   - 강의를 수강하는 고객들의 유형이 다양한 상황에서 환불, 수강, 주문 등 고객 정보와 관련된 데이터들을 통해 타사로의 고객 이탈 방지 요인들을 점검

- 수행역할
   - 데이터 수집, 전처리, EDA(탐색적 데이터 분석), 데이터 시각화, 데이터 타입 및 결측치 확인
   - 각 데이터별로 특징들을 분석
        - 매출이 가장 높은 날과 낮은 날, 주문수, 신청자수, 강의수
        - 월별 강의 할인율
        - 강의 키워드에서 가장 많이 언급된 단어
        - 환불이 가장 많은 강의
        - 가장 강의를 많이 신청한 고객과 그 금액
        - 주말과 주중 평균 강의 신청율 차이와 회원가입이 활발한 시간
        - 월별 강의 신청상태(완료, 진행중, 보류중)간 금액과 유저비중
   - 2022년 1월부터 12월 사이 각 데이터의 추세분석 및 시각화

- 기대효과 및 Lesson and Learned
  + 기대효과
    - 다양한 형태의 데이터 분석으로 강의 주문, 수강, 선호 요일 등 특징 파악이 가능하여 고객들의 패턴 파악 가능
    - 고객들의 전체적인 패턴파악으로 인해 환불 금액 감소, 타사 이탈 방지 등 대책 수립 가능

  + Lesson and Learned
    - 데이터 전처리의 중요성과 각 강의관련 데이터별 특징 이해
    - 여러가지 형태의 분석결과 도출로 고객 데이터 다룰 수 있는 능력향상
